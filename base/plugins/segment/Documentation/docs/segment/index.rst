Introduction
============

Ce logiciel est un éditeur pour SEGMent. Il se décompose en trois
parties:

- L’éditeur
- La bibliothèque
- L’inspecteur

La majorité des actions sont disponibles par glissé-déposer.

En cas de bug: contacter jeanmichael.celerier@gmail.com
